Changelog

This page summarizes the major releases and important changes for OpenGBSA. For a complete, detailed list of all changes, see the main ``CHANGELOG.md`` file in the repository.

Latest Release
--------------

**v0.0.2**

- Improved conda packaging and dependency pinning (Python, NumPy, OpenMM)
- Fixed critical IndentationErrors in runner scripts
- Overhauled documentation: new Sphinx portal, configuration guide, troubleshooting, outputs, contributing
- Added Material-style callouts and modern Furo theme
- Updated README and installation instructions
- Enhanced test coverage and validation
